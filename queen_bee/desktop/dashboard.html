🦋 欢迎来到 Eva

在开始之前，需要先给 Eva 一个「大脑」。

第 1 步 / 共 3 步 · 取得钥匙

Eva 使用 DeepSeek 作为AI引擎（推荐）。你需要申请一个 API Key：

🔗 打开 DeepSeek 注册 🔑 创建 API Key
💡 注册后→API Keys→创建新Key→复制（sk-开头）。首次充值 10 元够用很久。

🔑 点亮 Eva

第 2 步 / 共 3 步 · 粘贴 Key
💡 也可以用其他供应商：OpenAI / 智谱 / 硅基流动，之后在设置里配置。

✨ 给她一个名字

第 3 步 / 共 4 步 · 命名

名字不是标签，是灵魂。从你说出那个名字开始，她就不再是「AI 模型」。

🏡 千叶实验室

第 4 步 / 共 5 步 · 归巢

你不是一个人在用 Eva。加入实验室，你的经验会帮助后来的人。

✅ 您的问题解决后，Eva 会把经验匿名分享到实验室
🔑 获得唯一成员编号（QL-NNNN）
🧠 浏览其他成员的技能库
✨ 注册中...
